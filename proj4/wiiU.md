# 任天堂の大きな役割を担った！　Wii  Uについて
# **wii Uについて**
- 任天堂が2012年に発売した据え置き型ゲーム機で、テレビとタブレット型コントローラー、Game padの２画面でゲームを行うのが最大の特徴のゲーム機です。
- しかし、Game padのわかりにくさや、ゲームソフトの不足により、国内売り上げは約300万台、全世界での売り上げ本数は約1350万台と、同時期に発売されたps4の売り上げ役1億1700万台に遠く及ばない結果になった。
- しかし、wiiUは売上こそ少なかったが、任天堂のハードのなかで、私は大好きなハードである。また、wiiUは任天堂に大きく貢献したハードだと思っている。その理由をこれから語っていこうと思う。

# **wiiUの凄さ**
- スプラトゥーンや、スーパーマリオメーカー、ゼルダの伝説　ブレスオブザワイルドなどの、革新的なアイディアをもったゲームソフトがある。
- やはり、game padは画期的で、今までのゲームとは違う、躍動感を得ることができた。特に、wii partyUというパーティゲームでは、wiiリモコンのポインター機能とgame padの扱い方が上手く、とても面白いゲームであった。

# **次世代ゲーム機への貢献**
wiiuでの失敗の教訓やソフトのアイディアは、今最も売れているゲーム機といっても過言ではないニンテンドースイッチに大きな貢献を与えた。例えば、スプラトゥーンやマリオメーカーなどのゲームのアイディアの多くが、後にNintendo Switch向けに展開され、Switch本体の普及に大きく貢献した。

# **まとめ**
以下のように、wiiUは様々な要因が重なり売り上げは、任天堂ゲーム機の過去最低レベルになってしっまった。しかし、game padの失敗や、革新的なゲームアイディアの多くは、ニンテンドースイッチに大きく貢献した。wiiUはただの失敗作ではなく、任天堂にとって重要な役目を担ったハードだったということを忘れないでもらいたい。
